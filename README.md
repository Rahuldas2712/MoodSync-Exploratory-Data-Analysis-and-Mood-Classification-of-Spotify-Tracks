This project leverages Exploratory Data Analysis (EDA) to categorize and visualize the emotional landscape of music using the Spotify Features dataset. By implementing a custom classification algorithm in Python, the script analyzes key audio features—such as valence, energy, danceability, and acousticness—to map songs into distinct mood categories: Happy, Sad, Energetic, Calm, and Neutral.

Using Pandas for data orchestration and Matplotlib/Seaborn for visualization, the project uncovers insights into:

Mood Distribution: Quantifying the prevalence of different emotional profiles across the dataset.

Popularity Trends: Identifying which musical moods resonate most with listeners based on average popularity scores.

Feature Correlation: Visualizing the relationship between physical attributes (like energy and danceability) and their impact on a song's perceived mood.

Artist Performance: Ranking the top 10 artists based on their average popularity metrics.
